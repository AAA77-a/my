# 黄安德个人主页

## 项目介绍
这是一个科技风格的个人主页，展示黄安德同学的AI编程技能和商务数据分析与应用专业背景。

## 技术栈
- 纯静态HTML5 + CSS3 + JavaScript
- 字体：Roboto 和 Orbitron
- 响应式设计，适配不同屏幕尺寸

## 功能特点
- 科技风格的UI设计
- 粒子背景效果
- 打字动画效果
- 技能条动画
- 平滑滚动导航
- 移动端适配
- 项目卡片悬停效果

## 部署指南

### 步骤1：创建GitHub仓库
1. 登录GitHub
2. 创建一个新的仓库，命名为 `username.github.io`（其中 username 是你的GitHub用户名）
3. 克隆仓库到本地

### 步骤2：上传文件
将以下文件上传到GitHub仓库：
- index.html
- style.css
- script.js
- README.md

### 步骤3：部署到Cloudflare Pages
1. 登录Cloudflare
2. 导航到Pages
3. 点击"Create a project"
4. 选择GitHub作为连接源
5. 选择你的仓库
6. 配置构建设置：
   - 构建命令：无需构建命令
   - 构建输出目录：/（根目录）
7. 点击"Deploy site"

### 步骤4：访问网站
部署完成后，你可以通过Cloudflare Pages提供的URL访问你的个人主页。
